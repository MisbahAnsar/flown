# FlowMind

FlowMind is a Next.js app where a user gives one natural language instruction (for example, "summarize my GitHub notifications") and a 3-agent pipeline (Interpreter, Fetcher, Actor) executes it, logging every action to a Soroban smart contract on Stellar testnet.
